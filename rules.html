Welcome to the Geography Knowledge Game!

Game Rules:

Players must install App Artcodes on their smartphones. Nine links are provided in advance. Players find an artcode within the book that correspond to these links. By scanning an artcode and correctly answering the associated question, a location will be marked on the map. This continues until all seven questions are answered correctly and their locations marked, at which point the player hits the 'End Game' button to finish. The goal is to mark all seven locations on the map within 10 minutes to successfully complete the game.

Game Details:

  1. Players scan a start artcode which leads to an 'index.html' page containing the game rules. They scroll to the bottom and click 'Start Game'.
  2. On the 'index.html' page, there's a map interface, 'Restart Game' button, 'End Game' button, and 'View Current Clue' button.
  3. Map Interface: Each time a player clicks on a question link and answers correctly, that artcode location is marked on the map.
  4. Restart Game Button: If there's an unexpected situation requiring a game reset, players can use this button to start over.
  5. End Game Button: After following the process and answering the final question correctly, players can finish the game by pressing this button.
  6. View Current Clue Button: After starting the game, players should immediately click this to receive the first clue link, which leads them to find a book based on the address in the clue.

Tip: If the webpage is slow to load after clicking the 'View Current Clue' button, refresh the page for a quicker load.
