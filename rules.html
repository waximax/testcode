Welcome to the Geography Knowledge Game!

Game Rules:

The objective of this game is to enhance your knowledge about Nottingham's distinguished alumni through engaging questions related to their achievements and contributions. You have 10 minutes to complete all the challenges.

How to Play:

  1. Install the App: Players must install the "App Artcodes" on their smartphones.
  2. Start the Game: Scan the start artcode to open the 'Game Start Page' displaying the game rules, then scroll to the bottom and click 'Start Game'.
  3. Playing the Game: Use the 'View Current Clue' button to get the first clue which provides an address to find the next artcode in a book. Repeat this for all seven questions.
  4. Buttons:
    • Restart Game: Resets the game if needed.
    • End Game: Finish the game after all questions are correctly answered and marked on the map.

Tip: If the webpage is slow to load after clicking 'View Current Clue', try refreshing the page for quicker access.
